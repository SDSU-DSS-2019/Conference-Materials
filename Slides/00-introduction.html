What is the tidyverse?

  • Collection of R Packages that:
    • have common coding styles
    • work well together
    • work quickly as they are coded to run in C++
  • Can be easily extended to work with:
    • relational databases
    • spark
    • hadoop

How Do we get this?

Do This Once

install.packages("tidyverse")

Do this each time

library(tidyverse)
